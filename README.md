aws-high-level-sdk-csharp

Services under development:
* Amazon Simple Queue Service
* Amazon SimpleDB
* Amazon DynamoDB

Current Capabilities:
* Amazon DynamoDB
** GetItem
** PutItem
** UpdateItem
** DeleteItem
** BatchWrite (Put & Delete)
** Scan
