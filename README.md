Simple AWS SDK C#
=========================

A Simple AWS SDK written in C#

Services under development:
* Amazon DynamoDB
* Amazon SimpleDB
* Amazon Simple Storage Service
* Amazon Simple Queue Service

Current Capabilities:
* Amazon S3
  * FindObject
  * PutObject
  * GetObject
  * DeleteObject
* Amazon DynamoDB
  * GetItem
  * PutItem
  * UpdateItem
  * DeleteItem
  * BatchWrite (Put & Delete)
  * Scan
